Datasets

Our team has organized some available temporal datasets which have been used by related studies. A brief description of the datasets is presented.


Dataset Description Link
Autonomous Systems 9 graphs of Autonomous Systems (AS) peering information inferred from Oregon route-views between March 31 2001 and May 26 2001. Link
DBLP The dblp computer science bibliography provides open bibliographic information on major computer science journals and proceedings. Link
Enron E-mail network among employees of Enron between 1999 and 2003. It contains data from about 150 users and when they exchanged a message with another member of the organization. The corpus contains a total of about 0.5M message. Link
Flickr Sampled a set of Flickr users with edges representing mutual friendship and edge time-stamp the moment when the bidirectional contact was established. Set your own time granularity. Link
IMDB IMDB Datasets. Contain information for movie titles, director and writer information for all the movie titles, principal cast/crew for movie titles and IMDb rating and votes information for titles. Link, Link
Koblenz Network Collection The KONECT project has 1,326 network datasets in 24 categories. Some of the datasets: Networks: Karate club, Slashdot Zoo, Twitter followers, Yeast, arXiv hep-th etc. Link
Reality Commons Badge Dataset: The data contain the performance, behavior, and interpersonal interactions of participating employees at a Chicago-area data server configuration firm for one month. Friends and Family Dataset, Reality Mining Dataset and Social Evolution Dataset. Link
Yelp Contains 209,393 business data including location data, attributes, and categories. Contains full 8,021,122 review text data including the user_id that wrote the review and the business_id the review is written for. User data including the user's friend mapping and all the metadata associated with the user. Checkins on a business. Tips written by a user on a business. Tips are shorter than reviews and tend to convey quick suggestions. Contains 200,000 photo data including the caption and classification (one of "food", "drink", "menu", "inside" or "outside"). Link
Wiki-Talk User interaction networks of Wikipedia of 28 different languages. Nodes (orininal wikipedia user IDs) represent users of the Wikipedia, and an edge from user A to user B denotes that user A wrote a message on the talk page of user B at a certain timestamp. Link
